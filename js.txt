1. alert() Function:

Syntax: alert("message");
Example: alert("Hello World!");

2. Data Types:

String: Represents text data, enclosed in double or single quotes.
Example: "Hello", 'World!'

Number: Represents numerical values (whole numbers or decimals).
Example: 123, 4.56

Boolean: Represents logical values, either true or false.
Example: true, false

3. Variables:

Purpose: Store data for later use.
Syntax: var variableName = value; (older) or let variableName = value; (modern)
Example: let name = "Tuman";

4. prompt() Function:

Purpose: Displays a popup that prompts the user for input.
Syntax: prompt("message");
Example: let age = prompt("Enter your age:"); would ask for input and store it in the age variable.

5. .length:

Strings: .length returns the number of characters in a string.
Example:
var myString = "Hello, World!";
console.log(myString.length); // This will output 13

6. slice();

Strings: Extracts a section of the string and returns it as a new string.
Does not modify the original string.
Takes two optional parameters:
start: The index of the first character to include (inclusive). Default is 0 (beginning of the string).
end: The index of the character to exclude (not included). Default is the end of the string. Negative values count from the end.
Example:
var str = "Hello World!";
var newStr = str.slice(6, 11); // newStr will be "World!"
var newStr2 = str.slice(6); // newStr2 will be "World!"
var newStr3 = str.slice(1, -1); // newStr3 will be "ello, World"

7. Changing Casing in Text:
    7.1. .toUpperCase():

    Return value: It returns a new string with all lowercase characters converted to uppercase. The original string is not modified.
    Case sensitivity: It only affects lowercase characters (a-z). Uppercase characters (A-Z) and other characters like numbers, symbols, and whitespace remain unchanged.
    Immutability: Since JavaScript strings are immutable, you cannot directly change the original string using toUpperCase(). You need to assign the returned value to a new variable.
    Example:
    let originalString = "Hello, World!";
    originalString = originalString.toUpperCase();
    console.log(originalString); // Outputs: "HELLO, WORLD!"

    7.2. .toLowerCase():

    Return value: It returns a new string with all uppercase characters converted to lowercase. The original string remains unchanged.
    Case sensitivity: It only affects uppercase characters (A-Z). Lowercase characters (a-z) and other characters like numbers, symbols, and whitespace remain unchanged.
    Immutability: Like other string methods in JavaScript, .toLowerCase() doesn't modify the original string. You need to assign the returned value to a new variable to keep the lowercase version.
    Example:
    let originalString = "Hello, World!";
    originalString = originalString.toLowerCase();
    console.log(originalString); // Outputs: "hello, world!"
